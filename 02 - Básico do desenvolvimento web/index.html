Trabalhando localmente

Como é feito uso de HTML simples, não é necessário nenhum servidor, mas para criar um ambiente mais próximo da realidade de hospedagens, vamos fazer uso de um pequeno servidor HTTP, que vai nos permitir ter a mesma experiência de um website.

Para isso é necessário que sua máquina tenha Node.JS instalado e também o NPM (gerenciador de pacotes do Node).


      $ npm install
      $ npm start
Após isso, basta acessar a URL http://127.0.0.1:8080 no seu navegador.
